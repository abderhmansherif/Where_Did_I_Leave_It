# Where Did I Leave It
Where Did I Leave It is a simple application that helps you keep track of where you left your belongings. Whether it's your keys, wallet, or phone, this app allows you to quickly jot down the location of your items so you can easily find them later.

## Features
- Full History Tracking
- Item Tracking
- Last Seen State



## Technologies Used
- .NET 10
- Entity Framework Core
- PostgreSQL
- Clean Architecture
- Domain Driven Design (DDD)
- CQRS



## Getting Started
To get started with the Where Did I Leave It application, follow these steps:
1. Clone the repository: `git clone
2. Navigate to the project directory: `cd where-did-i-leave-it`
3. Install the dependencies: `dotnet restore`
4. Set up the database connection string in the `appsettings.json` file.
5. Run the application: `dotnet run`
6. Open your browser and navigate to `http://localhost:xxxx/scalar/v1` to access the API documentation.
